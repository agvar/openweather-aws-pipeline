# OpenWeather AWS Data Pipeline

A robust weather data collection and storage system using OpenWeatherMap API and AWS services.

## Features
- Weather data collection from OpenWeatherMap API
- AWS S3 storage integration
- Infrastructure as Code with Terraform
- Environment-based configuration

## Quick Start
1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up environment variables (see .env.example)
4. Run: `python src/WeatherDataCollector.py`

## Tech Stack
- Python 3.x
- AWS (S3, IAM)
- Terraform
- OpenWeatherMap API
